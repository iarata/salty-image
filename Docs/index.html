Alpha Blending

In computer graphics, the combining of the alpha channel with other layers in an image in order to show translucency. The alpha channel is an additional eight bits used with each pixel in a 32-bit graphics system that can represent 256 levels of translucency. Black and white represent opaque and fully transparent, while various gray levels represent levels of translucency. Source: Definition of alpha blending by PCMag

In our code however, we directly modify the RGB values to give an illusion of transparency. It is not truly transparent, since the image only has 3 channels, and not 4

Softness

Because the implementation of the gamma correction did not work, we are using softness instead, where you can adjust the softness of the edges of the two images. It is essentially a multiplier for the alpha value for alpha blending.
